Python code using Selenium WebDriver
====================================

A simple Python code tusing Selenium WebDriver.

Programming Language and libraries used to solve the problems
-------------------------------------------------------------

  - Python v3.6.1

Main libraries:

  - selenium v3.4.3
  
Running the code:
-----------------
- Install dependencies: 
   $ make init

- Run the simple test code: 
   $ make run_simple

- Run the class test:  
   $ make run_class
